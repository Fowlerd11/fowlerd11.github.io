Lunar Intrusion
Windows

Lunar Intrusion is a game I made with GameMaker: Studio, a popular and easy-to-use development suite for making video games. Intrusion is a platformer/arcade-shooter game intended for a middle school audience. Aliens start attacking a military base on the Moon, and it's the player's job to defend it with a stationary turret that fires missiles. The aliens fall out of dropships and float to the ground in an attempt to take the player out. Every kill earns you points, and the object of the game is to get the highest score possible.

I spent approximately 50 hours on this game. There was a lot of planning and problem solving that I had to work through, but it was well worth it in the end. I wanted the game to be like Sabotage on the Apple II, and I think I recreated it very well. It has a modern, flat design and the animations are very clean. The game features menus, a tutorial, and a cut-scene: all of which include some clean animations. The gameplay itself is very smooth with minimal bugs and it progressively gets harder as the game goes on.

Lunar Intrusion is compatible with Windows only.

Flappy Hawk
iOS + Android

Amidst Flappy Bird's insane success, a lot of remakes of the game were being made. I decided to make my own as well, but in a different way most developers were going. Flappy Hawk isn't a complete remake of Flappy Bird, it's just a custom theme for Flappy Bird that has some school spirit to it.

This was more of a Phototshop project than anything, and a little bit of extra work was done for the Android version. On iOS, the theme is available only to jailbroken devices through the BigBoss repo on Cydia. The theme can be enabled with WinterBoard.

For Android, I recompiled the original app with my own images and created my own installable version of Flappy Bird. You can download the .apk file here. Please note that you will need to enable Unknown Sources in your settings before the app will be able to install.

Red Riding Hood
Adobe Flash

This is an animation I made in Adobe Flash. It shows a small part of the story of Little Red Riding Hood. Red goes to her grandma's house and notices that grandma looks different. The wolf (dressed as her grandma) jumps out of bed and chases Red, then escapes the wolf and hides.

This project was my first experience with Adobe Flash. I spent a few hours on it, most of which were spent learning how to use the program. I wanted it to be a nice, clean animation that tells the story of Red and the wolf. I used Adobe Illustrator and Photoshop to create images that weren't pixelated and fit better with the house layout I created in Flash. I put some mysterious and puzzling music in the background, and it fit quite well.

You can see the animation here.

The Solar System (Alt)
Web

This is an alternate version of The Solar System. For this, I combined the two.js animation and the information from the original site. Hovering the mouse over a planet makes the planet's description show up in a box in the corner. The planets all move around the Sun at different speeds. It has the same audience and goals as the original website.

This was simply a side project I worked on for a little while. It's coded mostly in JavaScript and uses 2D objects. Every time the page is loaded, the stars are in different formations. I'm very happy with how this turned out for a beginner project with 2D objects.

The Solar System (Alt) validates under the W3C® HTML5 Conformance Checker. You can view the page here.

The Solar System
Web

The Solar System is a website designed to teach 3rd graders about our solar system. It has a short description of the Sun and the 8 different planets, talking about things like the number of moons, number of rings, number of days in a year, and other fun facts.

This was my first real project with web development, and I'm very proud of the finished product. It's a static website with well done CSS positioning. The site makes good use of JavaScript with features like drop-down menus, fade animations, and a 2D animation made with two.js. It transitions between pages seamlessly with the fade animations, making it seem like you're never opening a new page. It's very similar to a PowerPoint presentation.

The Solar System validates under the W3C® HTML5 Conformance Checker. You can view the website here.
